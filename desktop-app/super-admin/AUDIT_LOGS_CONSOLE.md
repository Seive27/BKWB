# Audit Logs Console - Live Monitoring

## Overview

The Audit Logs Console provides real-time monitoring of system events and security audit streams in a command-line interface style. This terminal-like view allows Super Admins to watch live log entries as they occur, with color-coded log levels and real-time statistics.

## File Location

- **Component**: `src/pages/AuditLogsConsole.tsx`
- **Modal Component**: `src/components/ExportAuditLogsModal.tsx`
- **Navigation**: Accessed via "Live Monitoring" button from Audit Logs page

### Helper Functions

#### getLevelColor
Returns appropriate Tailwind color class for log level:
```typescript
const getLevelColor = (level: string) => {
  switch (level) {
    case 'INFO': return 'text-blue-400';
    case 'WARN': return 'text-yellow-400';
    case 'ERROR': return 'text-red-400';
    case 'AUDIT': return 'text-green-400';
  }
};
```

#### getLevelLabel
Formats log level with consistent padding:
```typescript
const getLevelLabel = (level: string) => {
  return level.padEnd(5, ' ');
};
```

### WebSocket Connection
```typescript
useEffect(() => {
  const ws = new WebSocket('ws://localhost:8080/audit-logs/stream');
  
  ws.onmessage = (event) => {
    const logEntry = JSON.parse(event.data);
    setLogs((prev) => [...prev.slice(-50), logEntry]);
  };
  
  return () => ws.close();
}, []);
```

### Export Implementation
```typescript
const handleExport = async () => {
  const response = await fetch('/api/audit-logs/export', {
    method: 'POST',
```
